Project Overview

System Description

Web-based Hybrid Machine Learning System for Malware Analysis and Cyber-Physical Systems Behavioral Prediction

Technical Architecture

Static Analysis Engine

  • Algorithm: Random Forest (RF)
  • Input Format: .bytes files
  • Dataset: Microsoft Malware Classification Challenge 2015
    • 9 malware families
    • 21,741 malware samples
    • Feature extraction from binary files
  • Model Specifications:
    • Number of trees: 100
    • Max depth: 10
    • Feature importance threshold: 0.01
  • Weight in Hybrid System: 60%

Dynamic Analysis Engine

  • Algorithm: Support Vector Machine (SVM)
  • Input Format: .png images (malware visualization)
  • Dataset: Malimg Dataset
    • 25 malware families
    • 9,339 malware samples
    • Image-based feature extraction
  • Model Specifications:
    • Kernel: RBF (Radial Basis Function)
    • C parameter: 1.0
    • Gamma: 'scale'
  • Weight in Hybrid System: 40%

Hybrid Voting System

  • Integration Method: Weighted Ensemble
  • Decision Threshold: 0.5
  • Confidence Scoring: Probability-based
