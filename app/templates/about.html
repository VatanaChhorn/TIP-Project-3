System Overview

This web system allows you to scan files for malware using both static and dynamic analysis.

How It Works

  • Static Analysis (.bytes files): Uses Random Forest (RF) trained on Microsoft Malware Classification Challenge 2015 dataset.
  • Dynamic Analysis (.png images): Uses Support Vector Machine (SVM) trained on the Malimg dataset.
  • Hybrid Voting: Combines RF (60%) and SVM (40%) to give a final classification.

Datasets Used

  • Microsoft Malware Classification Challenge 2015
  • Malimg Dataset (Malware Images)
